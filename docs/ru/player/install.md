# Установка

На момент написания гайда, вы НЕ можете поиграть с этим модом без использования SteamVR из-за особенностей работы раскладок Vivecraft.

## Установка мода {#mod-install}

### Автоматическая установка {#auto}

Если вы используете лаунчеры для сборок (Modrinth App, Prism Launcher, Curseforge и другие), найдите и добавьте InteractiveMC в вашу сборку.

### Ручная {#manual}

1. Скачайте jar-файл InteractiveMC и его зависимостей в соответствии с нужной вам версией игры и загрузчиком:
- [InteractiveMC](https://modrinth.com/project/interactivemc)
- [Vivecraft](https://modrinth.com/mod/vivecraft)
- [Velthoric](https://modrinth.com/mod/velthoric)
- [Architectury API](https://modrinth.com/mod/architectury-api)

2. Переместите скачанные файлы в папку `mods` по пути `%appdata%/.minecraft/mods` (если вы не меняли пути установки вручную.)

## Настройка {#setting-up}

1. Откройте настройки SteamVR, выберите категорию `Контроллеры`, выберите `Управление раскладками контроллера`, при необходимости выберите `Vivecraft` в списке приложений.
![Manage Controller Settings](/assets/player/steamvr_settings_managecontrollerbindings.png)

2. В интерфейсе редактирования раскладок выберите вкладку `В игре` и задайте следующие назначения в соответствии со скриншотом:
- Триггер основной руки
- Касание триггера основной руки
- Хват основной руки
- Триггер второй руки
- Касание триггера второй руки
- Хват второй рукой
![Controller Bindings](/assets/player/steamvr_controllerbindings.png)
