# Гайды для игроков

Эта часть документации посвящена игрокам и администраторам серверов.

Воспользуйтесь боковой панелью для навигации по гайдам.

Вы всегда можете попросить помощи в [обсуждениях](https://github.com/Timtaran/InteractiveMC/discussions) на GitHub, а так же сообщить о проблеме/желаемой функции, используя [issue](https://github.com/Timtaran/InteractiveMC/issues).
