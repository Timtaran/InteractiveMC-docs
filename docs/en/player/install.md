# Installation

At the time of writing this guide, you cannot play with this mod without SteamVR because of the way Vivecraft bindings work.

## Installing the mod {#mod-install}

### Automatic installation {#auto}

If you use modpack launchers (Modrinth App, Prism Launcher, CurseForge, and others), find and add InteractiveMC to your pack.

### Manual installation {#manual}

1. Download the InteractiveMC jar file and its dependencies for the game version and loader you need:
- [InteractiveMC](https://modrinth.com/project/interactivemc)
- [Vivecraft](https://modrinth.com/mod/vivecraft)
- [Velthoric](https://modrinth.com/mod/velthoric)
- [Architectury API](https://modrinth.com/mod/architectury-api)
- [YetAnotherConfigLib](https://modrinth.com/mod/yacl)

2. Move the downloaded files to the `mods` folder at `%appdata%/.minecraft/mods` (if you did not change the installation path manually).

## Setup {#setting-up}

1. Open the SteamVR settings, select the `Controllers` category, choose `Manage controller bindings`, and if needed select `Vivecraft` from the application list.
![Manage Controller Settings](/assets/player/steamvr_settings_managecontrollerbindings.png)

2. In the binding editor interface, select the `In-Game` tab and assign the following actions according to the screenshot:
- Main-hand trigger
- Main-hand trigger touch
- Main-hand grab
- Off-hand trigger
- Off-hand trigger touch
- Off-hand grab
![Controller Bindings](/assets/player/steamvr_controllerbindings.png)
